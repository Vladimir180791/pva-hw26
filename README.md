# Process Monitor - Парсер системных процессов Linux

Парсер системных процессов ОС Linux, написанный на Python с использованием стандартной библиотеки и модуля subprocess.

## 📋 Описание

Скрипт анализирует системные процессы с помощью команды `ps aux` и предоставляет подробный отчет о:
- Пользователях системы
- Количестве запущенных процессов
- Использовании памяти и CPU
- Процессах с максимальным потреблением ресурсов
